Jokarus-Wiki
============

We try to gather all documentation here that relates to JOKARUS processes and
things. This will/should include

- Documentation off all python code, automatically extracted from the code
  itself (*API documentation*)
- Hints on how to use the software
- Information about existing Software in use for JOKARUS, along with how and why
  it is used
- Information related to all devices used in the experiment
- Details about the experimental design and decisions
